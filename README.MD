# Word List Generator

This program provides a comprehensive collection of categorized word lists, designed for creative naming, storytelling, world-building, or any other project that requires rich and diverse vocabulary. The words are organized into meaningful categories such as **environments**, **emotions**, **mythology**, **science**, **fantasy**, and more.

---

## Features

- **Categorized Word Lists**: Words are sorted into logical categories like `attributes` and `objects`, with subcategories for easy navigation.
- **Expandable**: The lists are designed to be easily expanded with new words or categories.
- **Versatile Use**: Suitable for generating names, creating fictional worlds, writing stories, or brainstorming ideas.
- **Python Code**: The word lists are provided in Python format, making them easy to integrate into scripts or applications.
